# sources/

Raw transcripts and extracted source text live here **locally only** — they're
third-party copyrighted material (lectures, books) and aren't republished.

What's public is the *tool* (`../ingest.py`), the *index* (the spine checklist
in the top-level README), and the hand-assigned `resonance` weights. Run the
ingester against your own copies to populate this folder.
