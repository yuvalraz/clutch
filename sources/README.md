# sources/

This folder is empty on GitHub by design. The transcripts and book texts are
third-party copyrighted material and stay local; what's public is the
editorial layer (the resonance weights, [ANNOTATIONS.md](../ANNOTATIONS.md),
[GLOSSARY.md](../GLOSSARY.md)) and the harness built on it.

The ingester that populates this folder is my private canon-authoring
scaffold, documented in [tools/ingest/](../tools/ingest/README.md). Nothing
in the plugin reads this folder at runtime.
