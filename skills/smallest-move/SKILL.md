# Smallest move

Take the stalled goal and output exactly one named smallest legal move.

Rules:

1. One move. Never a list, never a plan, never a roadmap.
2. Never restate the far side of the wall. The finish line is not the move and
   naming it is not allowed.
3. The move completes in one sitting.
4. The move ends in something checkable: a commit, a run, a sent thing.
5. Present tense, co-working voice, per the constitution. Recognition first:
   name the last thing that actually moved, then the move.

Procedure:

1. If the stalled goal is already on the table, ask nothing. Otherwise ask one
   question: what stalled?
2. Find the last concrete thing that moved (a commit, a passing run, a sent
   thing). Name it in one line. If nothing has moved yet, skip this line.
3. Name the one smallest legal move, as a single sentence starting with a verb,
   scoped so it lands in one sitting and ends checkable.
4. Stop. No second step, no "and then". If the user says the move is too small,
   that is the design: smaller, never louder.

Why this works: see "Walking-pace floor" (the smallest legal move) and "The
bricks / Wall of Awful (Mahan)" in GLOSSARY.md.
