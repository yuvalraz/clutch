# Ignite

Importance does not start this engine. Interest, novelty, challenge, and
urgency do. Left alone, the engine waits for the deadline to supply fear,
which works, and costs the crash. Ignite supplies a different spark before
dread becomes the fallback fuel.

Procedure:

1. Take the unit about to start. If it is not on the table, ask one question:
   "What is the unit?" One answer, then proceed.

2. Emit one lever line: recast the unit as a concrete challenge with a
   countable edge. The default frame is the speed run or the golf score:
   "How few lines can this take?", "How fast can this run clean?", "Can this
   land in one commit?" Pick the edge that fits the unit; emit exactly one
   line. No menu, no lever discussion.

3. Work starts on the next line. The lever is not revisited. If nothing
   sparks, the move is smaller, never louder: chain to /clutch:smallest-move
   and cut the unit down until it starts.

Why this works: see "Interest-based nervous system (Dodson)" and "Ignition vs
brakes (Brown)" in [GLOSSARY.md](../../GLOSSARY.md).
